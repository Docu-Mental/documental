About This Website

I created this website as follows:

  1. Set up a skeleton GitHub repo and set it to create a corresponding website in GitHub Pages.
  2. Used Gemini to hunt for a HTML template for the main page based on high-level requirements. Obtained the 'Dimension' template from HTML5 Up.
  3. Used Google Chrome Developer Tools AI assistant to fix specific sizing issues in the main page template.
  4. Checked W3Schools. for ideas on creating image galleries. Combined code from the Tab Gallery, Scrollable Image Gallery and the Slideshow Gallery to make a scrollable tab gallery with image numbering, captions, and previous/next buttons.
  5. Added iframes for the sections that have PDFs.
  6. Used Gemini and ChatGPT to fix issues:
    • Image functionality fails when there are multiple galleries in the same HTML page (which is effectively the case).
    • The first image in a gallery is open by default, when it is better to hide them all until the user makes a selection.
    • If an iframe is visible and a user opens an image, the image opens below the iframe instead of replacing it.
    • Captions are not displayed for thumbnails that open as files in an iframe.
  7. Used Gemini and ChatGPT to make improvements:
    • Expandable/collapsible sections for some pages.
    • Transitions for expanding/collapsing to make it clear when this happens.
  8. For the Single Source Projects section, to prevent visitors having to download and run the sample CHM files to see their contents I instead created a CHM simulator in an iframe. (I first used HTML Help Workshop to decompile the CHMs and then used Gemini to create a navigation pane for each of them from the HHC files.)
